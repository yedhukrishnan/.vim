```
Color Scheme: brogrammer
Font: Fira Code
```

### Installation

``` bash
git clone https://github.com/yedhukrishnan/.vim.git
cp .vim/vimrc .vimrc
```

Open vim and execute the following to install the plugins:

```
:PlugInstall
```
